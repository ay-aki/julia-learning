微分積分

微分積分学について説明します。 あまり厳密には扱いません。

微分と積分

微分と積分は、数学の基本的な操作であり、相互関係を持つ重要な概念です。 微分は関数の変化率を求める操作であり、積分は関数の面積や累積量を求める操作です。 以下にそれぞれの操作を説明し、相互関係についても触れます。

微分

微分（Differentiation）は、関数の微小な変化率や勾配を求める操作です。ある関数の微分は、その関数の各点での傾きを表します。微分は関数の変化を詳細に分析するために使用され、物理学や経済学などの科学的なモデリングにおいても重要です。

\[ \frac{d}{dx} F(x) = f(x) \]

左辺は以下の極限で表すこともできます。 実用上、とても大切な式です。

\[ \lim_{\delta x -> 0} \frac{f(x + \delta x) - f(x)}{\delta x} \]

積分

積分（Integration）は、関数の面積や累積量を求める操作です。積分は関数を積み上げることにより、元の関数が表す領域の面積や累積を求めます。積分は、物理学の運動方程式の解析や確率論の確率密度関数の計算などで使用されます。

\[ F(x) = \int_{c}^{x} f(t) dt \]

微分と積分の相互関係は、微分と積分の逆操作を表します。

微分と積分の関係

微分と積分は基本的な定理によって関連付けられています。 ニュートン・ライプニッツの基本定理によれば、ある関数f(x)の積分とその関数の原始関数F(x)の微分は互いに逆の関係にあります。

\[ \int_{a}^{b} f(x) dx = F(b) - F(a) \]

この定理により、積分は微分の逆操作として捉えることができます。

具体的には、関数f(x)を微分して得られる導関数f’(x)を求め、その導関数を積分すると元の関数f(x)が得られます。

\[ \frac{d}{dx} f(x) = f'(x) \]

\[ \int f'(x) dx = f(x) + C \]

ここで、Cは積分定数です。微分と積分の逆操作によって、元の関数を復元することができます。

このように、微分と積分は相互に関連しており、微分によって関数の変化率や勾配を求め、積分によって関数の面積や累積量を求めることができます。微分と積分は数学の基礎であり、物理学、経済学、工学、統計学などのさまざまな分野で広く応用されています。

数値積分、数値微分は数値解析の説明に譲ります。

微分公式

それぞれの微分公式をJuliaで確認しましょう。

SymPyでは微分積分などを扱えますが、対応していない計算に対しては動作しないので過信しないでください。

\(f(x)=x^n\)

微分すると、 \(f'(x)=nx^{n-1}\) となる。 これをJuliaで書くと以下のようになる:

using SymPy
@vars x
diff(x^10, x)

\(10 x^{9}\)

\(f(x)=\sin(x), g(x)=\cos(x)\)

微分すると、 \(f'(x)=\cos(x), g'(x)=-\sin(x)\) となる。 これをJuliaで書くと以下のようになる:

display.(
    (diff(sin(x), x), diff(cos(x), x))
);

\(\cos{\left(x \right)}\)

\(- \sin{\left(x \right)}\)

\(f(x)=e^x\)

微分すると、 \(f'(x)=e^x\) となる。 これをJuliaで書くと以下のようになる:

diff(exp(x), x)

\(e^{x}\)

\(f(x)=a^x\)

微分すると、 \(f'(x)= a^x \log(a)\) となる。 これをJuliaで書くと以下のようになる:

diff(10^x, x)

\(10^{x} \log{\left(10 \right)}\)

\(f(x)=e^x\)

微分すると、 \(f'(x)=e^x\) となる。 これをJuliaで書くと以下のようになる:

diff(exp(x), x)

\(e^{x}\)

\(f(x)=\log(x)\)

微分すると、 \(f'(x)=\frac{1}{x}\) となる。 これをJuliaで書くと以下のようになる:

diff(log(x), x)

\(\frac{1}{x}\)

\(f(x)=\sinh(x), g(x)=\cosh(x)\)

微分すると、 \(f'(x)=\cosh(x), g'(x)=\sinh(x)\) となる。 これをJuliaで書くと以下のようになる:

display.(
    (diff(sinh(x), x), diff(cosh(x), x))
);

\(\cosh{\left(x \right)}\)

\(\sinh{\left(x \right)}\)

関数の和 \(h(x)=(f+g)(x)\)

微分すると、 \(h'(x)=(f'+g')(x)\) となる。 これをJuliaで書くと以下のようになる:

f = sympy.Function("f")(x)
g = sympy.Function("g")(x)
diff((f + g)(x), x)

\(\frac{d}{d x} f{\left(x \right)} + \frac{d}{d x} g{\left(x \right)}\)

関数の積 \(h(x)=f(x)g(x)\)

微分すると、 \(h'(x)=(f'*g)(x)+(f*g')(x)\) となる。 これをJuliaで書くと以下のようになる:

f = sympy.Function("f")(x)
g = sympy.Function("g")(x)
diff((f * g)(x), x)

\(f{\left(x \right)} \frac{d}{d x} g{\left(x \right)} + g{\left(x \right)} \frac{d}{d x} f{\left(x \right)}\)

合成関数 \(h(x)=f(g(x))\)

微分すると、 \(h'(x)=f'(g(x))g'(x)\) となる。 これをJuliaで書くと以下のようになる:

f = sympy.Function("f")(x)
g = sympy.Function("g")(x)
diff(f(g(x)), x)

\(\frac{d}{d g{\left(x \right)}} f{\left(g{\left(x \right)} \right)} \frac{d}{d x} g{\left(x \right)}\)

例えば……合成関数 \(f(x)=\sin(x^{10})\)

微分すると、 \(f'(x)=10x^9 cos(x^{10})\) となる。 これをJuliaで書くと以下のようになる:

diff(sin(x^10), x)

\(10 x^{9} \cos{\left(x^{10} \right)}\)

積分公式

それぞれの積分公式をJuliaで確認しましょう。

なお、今回は積分定数を無視します。

\(f(x)=x^n, g(x)=\frac{1}{x}\)

積分すると、\(F(x)=\frac{1}{n+1}x^{n+1}, G(x)=\log(|x|)\) となる。 これをJuliaで書くと以下のようになる:

using SymPy
@vars x
display.(
  (integrate(x^10, x), integrate(1/x, x))
)
;

\(\frac{x^{11}}{11}\)

\(\log{\left(x \right)}\)

\(f(x)=\sin(x), g(x)=\cos(x), h(x)=\tan(x)\)

積分すると、\(F(x)=-\cos(x), G(x)=\sin(x), H(x)=-\log(|\cos(x)|)\) となる。 これをJuliaで書くと以下のようになる:

display.(
    (integrate(sin(x), x), integrate(cos(x), x), integrate(tan(x), x))
)
;

\(- \cos{\left(x \right)}\)

\(\sin{\left(x \right)}\)

\(- \log{\left(\cos{\left(x \right)} \right)}\)

\(f(x)=e^x\)

積分すると、\(F(x)=e^x\) となる。 これをJuliaで書くと以下のようになる:

integrate(exp(x), x)

\(e^{x}\)

\(f(x)=\log(x)\)

積分すると、\(F(x)=x \log(x) - x\) となる。 これをJuliaで書くと以下のようになる:

integrate(log(x), x)

\(x \log{\left(x \right)} - x\)

\(f(x)=\sinh(x), g(x)=\cosh(x)\)

積分すると、\(F(x)=\cosh(x), G(x)=\sinh(x)\) となる。 これをJuliaで書くと以下のようになる:

display.(
    (integrate(sinh(x)), integrate(cosh(x)))
)
;

\(\cosh{\left(x \right)}\)

\(\sinh{\left(x \right)}\)
