Julia入門「アルゴリズムと数値計算を極める！」

Juliaのパッケージ

インストールするには、JuliaのREPLでusing Pkg; Pkg.add("DataFrames")を実行してください。 使いたいタイミングでusing DataFramesと指定して読み込んでください。

using DataFrames

なおデータフレームを使うパッケージとして、juliaにはPandasというパッケージもあります。

DataFramesとは？

データフレームとは、主にPythonのpandasというライブラリで使われるデータの形式です。 特に機械学習で使われているイメージです。

が、余り高速ではないので、本当に高い効率を必要とするなら、普通に行列を使ってください。

表を模したデータ構造で、一般的に 行(row)方向にはデータの系列を説明する特徴量(feature)が並び、 列(column)方向には特徴を表現するデータ(index)が並びます。

sepallength sepalwidth petallength petalwidth class
5.1 3.5 1.4 0.2 Iris-setosa
4.9 3.0 1.4 0.2 Iris-setosa
… … … … …

仕様が頻繁に変更される場合があることは注意しておいてください。

DataFrameコンストラクタ

空のデータを生成する

DataFrameはデータフレームを表す型で、コンストラクタにより以下のように生成できる。

df = DataFrame()
0×0 DataFrame

行列から生成する。

特徴名feature_nameはStringでもSymbolでも大丈夫です。 この記事ではSymbolとします（SymbolはExprが包含する変数を表す型だと思ってください）。

特徴名を自動生成したい場合、以下の第二引数を:autoにしてください。

A = [
  1 2.0 "a";
  3 4.0 "b";
  5 6.0 "d";
  7 8.0 "d";
  9 0.0 "e"
]
feature_name = [:num_odd, :num_even, :alphabet]
df = DataFrame(A, feature_name)
5×3 DataFrame
Row num_odd num_even alphabet
Any Any Any
1 1 2.0 a
2 3 4.0 b
3 5 6.0 d
4 7 8.0 d
5 9 0.0 e

辞書から生成する。

表記があれですが、上と似た結果になってるのが分かると思います。

d = Dict(
  :num_odd  => [1  , 3  , 5  , 7  , 9  ], 
  :num_even => [2.0, 4.0, 6.0, 8.0, 0.0], 
  :alphabet => ["a", "b", "c", "d", "e"]
)

直接生成する。

DataFrameの内部の可変長引数の仕組みを使って直接生成することもできます。

df = DataFrame(
  :num_odd  => [1  , 3  , 5  , 7  , 9  ], 
  :num_even => [2.0, 4.0, 6.0, 8.0, 0.0], 
  :alphabet => ["a", "b", "c", "d", "e"]
)

Int64 Float64 String
1 1 2.0 a
2 3 4.0 b
3 5 6.0 c
4 7 8.0 d
5 9 0.0 e

以下、以下の乱数生成されたデータフレームを参考に記します。

using Random
Random.seed!(0)
df = DataFrame(
  round.(rand(100, 10), digits=3), 
  :auto
)
;

データの追加と特徴の追加

行方向にデータが、列方向に特徴があるのはもう説明しましたね。

データの追加は、push!かvcat関数を用いて行い、特徴の追加は、hcatを用いて行ってください。 ここでは、データの追加をpush!で、特徴の追加をhcatを使ってやる方法を示します。

以下が元のデータフレームです。

df1 = DataFrame(rand(5, 3), :auto)
5×3 DataFrame
Row x1 x2 x3
Float64 Float64 Float64
1 0.160143 0.774666 0.624384
2 0.501136 0.781732 0.0565774

5 0.655636 0.409758 0.524008

SQLライクな操作

DataFramesの機能を用いてSQLっぽい操作ができます。

select

# select x1, x4, x5 as a
df_select = select(df, :x1, :x4, :x5 => :a)
first(df_select, 5)
