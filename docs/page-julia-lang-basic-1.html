関数の定義

関数の作り方1function ~ end

function f1(x, y)
  println("calc: $x ^ $y")
  return x ^ y
end
f1(5, 3)
calc: 5 ^ 3
125

関数の作り方2f(...)=...

f2(x, y) = x ^ y
f2(5, 3)
125

関数の作り方3f = (...) -> (...)

f3 = (x, y) -> x ^ y
f3(5, 3)
125

キーワード変数function f(...;...)

※文字列を生成した後に表示しているため、表示の順序に注意

function f4(x, y; c=1)
  println("calc: $c * $x ^ $y")
  return c * x ^ y
end
println("$(f4(5, 3)) $(f4(5, 3, c=2))")
calc: 1 * 5 ^ 3
calc: 2 * 5 ^ 3
125 250

ブロードキャスト

ベクトルと定数の間の四則演算

他の計算も大抵同じことができます

v, a = [1.0, 2, 3], 3

v1   = v .+ a # それぞれの要素に足し算
v2   = v .- a # 　　　　　　　　引き算
v3   = v .* a # 　　　　　　　　掛け算
v4   = v ./ a # 　　　　　　　　割り算

println(v1, v2, v3, v4)
[4.0, 5.0, 6.0][-2.0, -1.0, 0.0][3.0, 6.0, 9.0][0.3333333333333333, 0.6666666666666666, 1.0]

関数に対しても……

# f(x) = x^2
function f(x)
  return x ^ 2
end
println(f.([1, 2, 3, 5]))
[1, 4, 9, 25]

2変数関数に対しても……

f(x, y) = x^y

l1 = f.([1, 2, 3, 4], 2)
l2 = f.(2, [1, 2, 3, 4])

display.([l1', l2']);
