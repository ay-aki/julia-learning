Juliaのパッケージ

インストールするには、JuliaのREPLでusing Pkg; Pkg.add("Images")を実行してください。 使いたいタイミングでusing Imagesと指定して読み込んでください。

using Images

Imagesとは？

Imagesはjuliaの画像処理を扱うパッケージです。

画像データの読み込み、保存、変換、操作など、さまざまな画像処理タスクをサポートしています（ゲームなどで画像を扱いたいときは別のライブラリを使ってください）。

高速な画像処理を実現するために、Juliaの高性能な数値計算機能や並列処理機能を活用しています。

Images.jlは、画像データを表現するためのImage型や、画像処理関数の集合などの主要な要素から構成されています。以下に、Images.jlの主な機能と使用例をいくつか示します。

ImageViewやImageShowというパッケージを併せて使えば、imshowなどの関数によって画像の表示ができますが、保存やnotebookでの表示は成功するので気にしないでください（このパッケージ、なぜか重いので私の環境では使っていません）。

画像の確認はdisplayで行ってください。

Imagesの使い方

画像の読み込み

画像はload関数で読み込めます。

img1 = load("tmp/__imagesample1.bmp")
display(img1)

画像の書き込み（保存）

書き込みはsave関数です。

x = collect(1.0:-0.1:0)
img2 = Gray.(x * x')
display(img2)
save("tmp/__imagesample2.png", img2)

画像の操作

グレイスケールにする。

img1_gray = Gray.(img1)

サイズを変更する。

siz = (10, 10)
img1_resized = imresize(img1, siz)

回転する。

img1_rotated = imrotate(img1, 45)

切り取る。

ss = size(img1)
img1_cropped = img1[1:ss[1] ÷ 2, 1:ss[2]  ÷ 2]
