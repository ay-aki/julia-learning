Juliaのパッケージ

※Genについては他のサイトを作ってそこで説明します。

インストールするには、JuliaのREPLでusing Pkg; Pkg.add("Gen")を実行してください。 使いたいタイミングでusing Genと指定して読み込んでください。

using Gen

Genとは？

MITのグループが開発した、オープンソースの生成モデルと確率的推論のためのパッケージです。 一部、DSL的な記述があります。 できることが多いので、何から手を付けたらいいやら……。

  • Genは確率的推論アルゴリズムの細かい実装を自動化する。

    トリッキーな数学や低レベルの実装を自動化しながら、ユーザのモデルに併せた効率的な確率的推論アルゴリズムを書くためのブロックを提供します。

  • Genはパフォーマンスのトレードオフを柔軟に操作することを可能としています。

    アルゴリズムの一部はユーザの指定する言語やアルゴリズムに置き換えることができます。

  • Genはカスタムハイブリッド推論アルゴリズムをサポートします。

    ニューラルネットワークによる計算（低コスト）とモデルベースの計算（高コスト）の両方の長所を生かしたアルゴリズムをサポートしています。

  • コンパイラを拡張することなく、カスタム推論アルゴリズムを書くことができる。

  • 確率的構造を持つモデルにおける効率的な推論

生成関数

Genの@genマクロはDSL（ドメイン固有言語）的な特殊な記述を可能とします。 導入される記述として、以下のような式があります。

  • {addr} ~ distribution(parameters)

    この式は、addrというアドレスに対して、ランダムチョイスを生成する確率分布を割り当てることを表しています。

    ここで言うアドレスとは、何らかの名前として使う値であり、任意で構いません（多くのサンプルでは、アドレスにSymbolやTupleを使っています）。

生成関数（関数を生成する関数）は以下のように@genマクロを用いて記述できます。 {addr} ~ distribution(parameters)と記述した部分は、値を生成してあとで使用されます。

ここで作る関数は単体ではほとんど意味がなく、変数やパラメータの分布について示しただけのものです。

using Plots, Random; Random.seed!(0)

@gen function line_model(xs)
  # 傾き(slope)と切片(intercept)の生成
  slope     = {:slope}     ~ normal(0, 1) # slope    : mean=0 v=1 の正規分布
  intercept = {:intercept} ~ normal(0, 2) # intercept: mean=0 v=2 の正規分布
  # 生成する関数: y(x) = a x + b
  y(x)      = intercept + slope * x
  # y: mean=y(xs) v=0.1 の正規分布
  for (i, x) = enumerate(xs)
    {(:y, i)} ~ normal(y(x), 0.1)
  end
  return y
end

事後推論

※制作中

function do_inference(model, xs, ys, amount_of_computation)
  # ysによりyのとる乱数の値（ランダムチョイス）を観測値に固定する
  observations = Gen.choicemap()
  for (i, y) = enumerate(ys)
    observations[(:y, i)] = y
  end
  # 観測値を用いてtraceっぽいものを作成
  trace, _ = Gen.importance_resampling(
    model, (xs,), observations, amount_of_computation
  )
  return trace
end
