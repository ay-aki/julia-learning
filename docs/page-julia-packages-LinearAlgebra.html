Juliaのパッケージ

インストールしなくても最初から用意されているパッケージです。 プログラムの最初の方でusing LinearAlgebraと指定して読み込んでください。

LinearAlgebraとは？

行列の計算などをするのに適したパッケージです。 ただし、疎行列などの問題に適したアルゴリズムではないので注意してください。

数値計算で使われる機能は大半このパッケージに含まれます。 また、BLASやLAPACKの機能が含まれます。

ここで述べる学問領域は僕の得意分野ですが、このセクションでは簡単な使い方の例を紹介するに留めます。

参考

基本的な機能

ノルム

ベクトルのノルムを求められます。 デフォルトはユークリッド空間における通常のノルム(２ノルム)になります。

using LinearAlgebra
norm([1, 2, 1])
2.449489742783178

内積

ベクトル同士のドット積（内積）を求められます。

dot([1,3,2], [4,1,6])
19

外積

ベクトル同士のクロス積（外積）を求められます。

cross([1,3,2], [4,1,6])
3-element Vector{Int64}:
  16
   2
 -11

行列式

行列の行列式を求められます。

A = [
    1 2 4;
    5 4 1;
    9 2 6.0
]
det(A)
-124.0

トレース

行列のトレースを求められます。

A = [
    1 2 4;
    5 4 1;
    9 2 6.0
]
tr(A)
11.0

固有値問題

固有値問題を解きます。

A = rand(10, 10)

F = eigen(A) # Λ, V = eigen(A)でもOK
Λ, V = F.values, F.vectors

norm(
    A*V - V*diagm(Λ)
)
1.3988919810697231e-14

特異値分解

特異値分解を解きます。(エコノミーサイズです)

A = rand(10, 8)

F = svd(A) # U, S, V = svd(A)でもOK
U, S, V = F.U, F.S, F.V

norm(
    A - U*diagm(S)*V'
)
3.2209424127120038e-15

LU分解

LU分解を解きます。

A = rand(10, 8)

F = lu(A) # L, U = lu(A)
L, U, p = F.L, F.U, F.p

norm(
   A[p,:] - L*U
)
3.9852227955858445e-16

QR分解

QR分解を解きます。

A = rand(10, 8)

F = qr(A)
Q, R = F.Q, F.R

norm(
    A - Q*R
)
1.3076087352977526e-15
