整数論

漸化式

漸化式は僕の分野だと何に役立つか良く分かりません。 同次形の微分方程式にちょっと似てるかなってくらいです。

ただ、再帰的な解法を解くためのツールとして、何かに役立つかもしれないと思います。

Naiveな方法なら、初期項 \(a_0\) から 更新式 \(a_n = f(a_{n-1}, ..., a_0)\) で逐次求めていけば良いだけのものです。

漸化式の解法

ケースA

A \(a_n = a_{n-1} + d\)  \(\Rightarrow\)  A’ \(\Delta a_n = a_n - a_{n-1} = d\)  \(\Rightarrow\)  \(a_n = nd + a_0\)

ケースB

B \(a_n = r a_{n-1}\)  \(\Rightarrow\)  \(a_n = r^n a_0\)

ケースC

C \(a_n = a_{n-1} + f(n)\)  \(\Rightarrow\)  C’ \(\Delta a_n = a_n - a_{n-1} = f(n)\)  \(\Rightarrow\)  \(a_n= a_0 + \sum_{k=1}^{n} f(k)\)

ケースD

D \(a_n = p a_{n-1} + q\)  \(\Rightarrow\)  D’ \(b_n = p b_{n-1}\)  \(\Rightarrow\)  B
式変形

\(p \neq 1\) かつ \(n \rightarrow \infty\) において \(a_n \rightarrow \alpha\) とする。 このとき、\(b_n = a_n - \alpha\) と置くと、以下のように変形できます。 \[ \begin{array}{rr} & a_n = p a_{n-1} + q\\ +\big{)} & \alpha = p \alpha + q\\ \hline & b_n = p b_{n-1} \end{array} \]

ケースE

E \(a_n = p a_{n-1} + q r^n\)  \(\Rightarrow\)  E’ \(b_n = \frac{p}{r} b_{n-1} + q\)  \(\Rightarrow\)  D
式変形

\(b_n=\frac{a_n}{r^n}\) と置くと、以下のように変形できます。 \[ \frac{a_n}{r^n} = \frac{p a_{n-1}}{r^n} + q \Rightarrow \frac{a_n}{r^n} = \frac{p}{r} \frac{a_{n-1}}{r^{n-1}} + q \Rightarrow b_n = b_{n-1} + q \]

ケースF

F \(a_n = p a_{n-1} + q n + r\)  \(\Rightarrow\)  F’ \(b_n = p b_{n-1} + q\)  \(\Rightarrow\)  D
式変形

\(b_n=a_n - a_{n-1}\) と置くと、以下のように変形できます。 \[ \begin{array}{rr} & a_n = p a_{n-1} + q n + r\\ +\big{)} & a_{n-1} = p a_{n-2} + q (n-1) + r\\ \hline & b_n = p b_{n-1} + q \end{array} \]

ケースG

G \(a_n + p a_{n-1} + q a_{n-2} = 0\)  \(\Rightarrow\)  特性方程式  \(\Rightarrow\)  \(t^n + p t^{n-1} + q t^{n-2} = 0, a_n=t^n\)  \(\Rightarrow\)  \(t^2 + p t + q = 0, a_n=t^n\)

ケースH1

H1 \(a_n^k = r a_{n-1}^l\)  \(\Rightarrow\)  H1’ \(k b_n = \log(r) + l b_{n-1}\)  \(\Rightarrow\)  D
式変形

\(b_n = \log(a_n)\)とすると、以下のように書き表せる。 \[k \log(a_n) = \log(r) + l \log(a_{n-1})\] \[k b_n = \log(r) + l b_{n-1}\]

ケースH2

H2 \(a_n^k = r a_{n-1}^l a_{n-2}^m\)  \(\Rightarrow\)  対数

ケースI

I \(a_n = \frac{p a_{n-1}}{r a_{n-1} + s}\)  \(\Rightarrow\)  逆数

ケースJ

J \(a_n = \frac{p a_{n-1}}{r a_{n-1} + s}\)  \(\Rightarrow\)  \(x=\frac{p x}{r x + s}\) で平行移動

ケースK

K \(a_n=p a_{n-1}+q b_{n-1} , b_n = r a_{n-1} + s b_{n-1}\)  \(\Rightarrow\)  行列の固有値
