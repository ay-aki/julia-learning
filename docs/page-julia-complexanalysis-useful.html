複素解析

複素数の便利さ

あ、この記事は数学の振りした物理とかそっち系の記事だと思ってください。 厳密さが要求される場合、複素数の世界での準備はかなり大変なので、やりません。

三角関数と指数関数

複素数の世界だと、指数関数は三角関数と同じです。

このことが、途轍もなく複素数を便利なものだと実感させてくれます。

まずは、以下の重要な公式（オイラーの公式）があります。 この公式は要は、２次元の実平面 \(\mathbb{R}^{2}\) で単位円上の点の座標を \(\begin{bmatrix}\cos \theta \\ \sin \theta \end{bmatrix}\) とするのと同じようにして、複素平面上ではこのように表せるよ……って式なのです。

ちなみに、\(i\) は虚数単位です。お忘れなく。

\[ e^{i \theta} = \cos \theta + i \sin \theta \]

これを覚えておけば（当然、指数の知識は要りますが）、面倒くさいサインコサインの公式、微分の公式、積分の公式、全部一瞬で導出できるようになります。

微分

三角関数の微分は、指数関数の微分から求められます。 幾何学的解釈として、位相を90度ずらすのが微分だと言うこともできます。

\[ \frac{d}{d \theta} e^{i \theta} = i e^{i \theta} = i (\cos \theta + i \sin \theta) = - \sin \theta + i \cos \theta \]

変形すると、以下のようになります。

\[ \frac{d}{d \theta} \cos \theta + i \frac{d}{d \theta} \sin \theta = - \sin \theta + i \cos \theta \]

係数比較すると、以下のようになります。 \[ \frac{d}{d \theta} \cos \theta = - \sin \theta, \;\;\; \frac{d}{d \theta} \sin \theta = \cos \theta \]

積分

積分はその逆です。

\[ \int e^{i \theta} d \theta= \frac{1}{i} e^{i \theta} = - i e^{i \theta} = - i (\cos \theta + i \sin \theta) = \sin \theta - i \cos \theta \]

変形すると、以下のようになります。

\[ \int \cos \theta d \theta + i \int \sin \theta d \theta = \sin \theta - i \cos \theta \]

比較すると、以下のようになります。 \[ \int \cos \theta d \theta = \sin \theta, \;\;\; \int \sin \theta d \theta = -\cos \theta \]

加法定理

\[ e^{i (\theta + \phi)} = \cos (\theta+\phi) + i \sin (\theta + \phi) \]

\[ e^{i (\theta + \phi)} = e^{i \theta} e^{i \phi} = (\cos \theta + i \sin \theta)(\cos \phi + i \sin \phi) = (\cos \theta \cos \phi - \sin \theta \sin \phi) + i (\cos \theta \sin \phi + \sin \theta \cos \phi) \]

比較すると、以下のようになります。 \[ \cos (\theta+\phi) = \cos \theta \cos \phi - \sin \theta \sin \phi, \;\;\; \sin (\theta + \phi) = \cos \theta \sin \phi + \sin \theta \cos \phi \]

三角関数の性質の式

三角関数の周期について、ちゃんと覚えてないと、\(\frac{\pi}{2}\) だけ位相をずらすとどうなるっけ？……的な疑問が出てくることがあります。

複素数の式を使えば、すぐに導出できます。

\[ e^{i (\theta + \frac{\pi}{2})} = e^{i \frac{\pi}{2}} e^{i \theta} = i e^{i \theta} = - \sin \theta + i \cos \theta \]

よって、比較すると以下のようになります。

\[ \cos (\theta + \frac{\pi}{2}) = - \sin \theta, \;\;\; \sin (\theta + \frac{\pi}{2}) = \cos \theta \]

ここで、\(i = e^{i\frac{\pi}{2}}\) を使っていますが、これは、複素平面上で、\(\frac{\pi}{2}\) 角度を曲げると \(i\) になるというだけの式です。

他の角度も以下のように、複素数で書けます。

\(e^{i \frac{\pi}{2}}=i\), \(e^{i \frac{\pi}{3}}=\frac{1 + \sqrt{3}i}{2}\), \(e^{i \frac{\pi}{4}}=\frac{1+i}{\sqrt{2}}\) など。
